# Prime Game - 質數遊戲

![Version](https://img.shields.io/badge/version-1.0.0-blue)
![License](https://img.shields.io/badge/license-MIT-green)

這是一款由 **奧羅桌遊設計工作室** 開發的網頁益智遊戲。遊戲核心結合了 **質數數列規則** 與圖論中的 **漢米爾頓路徑 (Hamiltonian Path)** 概念，旨在挑戰玩家的邏輯推理與直覺。

## 🎮 遊戲玩法

在一個 $N \times N$ 的棋盤中，數字從指定起始值開始，依序連成一條不重複經過任何格子的路徑（漢米爾頓路徑）。

- **一般規則**：玩家必須依序填入「合成數」。如果下一個數字是「質數」，則該質數會預先出現在棋盤上，玩家填入的合成數必須同時與「前一個數字」和「下一個質數」相鄰。
- **目標**：從起始數字開始，順利串連所有格子至最大數字。

## 🌟 核心功能

### 1. 多樣化遊戲模式
- **基本模式**：從 1 開始，提供 5 階、7 階、10 階不同難度等級。
- **困難模式**：起始數字從 105 開始。由於數字越大，質數密度越低，挑戰難度顯著提升。
- **隱藏模式**：只顯示質數的位置而不顯示其數值，挑戰玩家對質數位置的直覺與數字序列的掌握。

### 2. 自由自定義 (Free Mode)
- **自訂尺寸**：支援從 $2 \times 2$ 到 $20 \times 20$ 的任意棋盤尺寸。
- **自訂起始值**：可任意設定數列的起始號碼（最大 999）。
- **提示與輔助**：
  - **提示系統 (Hints)**：內建檢查機制，可即時高亮填寫錯誤的格子。
  - **顯示答案**：自由模式下可一鍵揭曉完整路徑（僅限自由模式）。
  - **暫停功能**：可暫停計時與輸入（僅限自由模式）。
- **匯入/匯出**：
  - **JSON 匯出**：可將生成的題目儲存為 JSON 檔案。
  - **JSON 匯入**：可載入已匯出的題目繼續挑戰。
  - **PNG 輸出**：可將棋盤當前的佈局輸出為圖片，方便列印或分享作為紙筆遊戲。

### 3. 全球排行榜
- 整合 **Firebase Firestore** 實現跨平台的成績存儲。
- 提供兩種排行榜：
  - **時間榜**：依完成時間（毫秒）排序，同時間者比較取消次數。
  - **取消榜**：依取消次數排序，同次數者比較完成時間。
- **Google 登入**：需使用 Google 帳號登入才能遊玩並提交成績（排行榜瀏覽可匿名）。

### 4. 遊戲介紹頁面 (About)
- 提供完整的遊戲規則說明。
- 包含 4×4 示意棋盤，直觀展示漢米爾頓路徑。
- 介紹漢米爾頓路徑的數學背景（圖論、NP-完全問題）與質數的角色。

## 🛠 技術架構

- **前端核心**：Vanilla JavaScript (ES6+), HTML5, CSS3。
- **資料存取**：使用 **Firebase 12.x**（透過 Google CDN 載入）進行即時資料庫讀寫。
- **認證**：Firebase Authentication，支援 Google 登入（遊玩必要）與匿名登入（排行榜瀏覽）。
- **資料庫安全規則**：Firestore Security Rules 確保只有 Google 登入的使用者能寫入成績。
- **路徑演算法**：
  - 核心採用 **Backbite Algorithm**。
  - 系統首先生成一個蛇形路徑，依棋盤大小執行 `20 × R × C` 次隨機 `backbite` 拓撲變換，確保生成的漢米爾頓路徑具有高度隨機性且保證每局皆有解。

## 🚀 快速開始

本專案為純靜態網頁架構，無需複雜的後端環境。

1. 下載或 Clone 本專案。
2. 直接雙擊 `index.html` 即可在瀏覽器中開始遊戲。
3. 遊玩前需使用 **Google 帳號登入**，並設定排行榜顯示名稱（最多 10 字）。
